# Greek Empire — Design Gallery

Design gallery page for [Greek Empire](https://greek-empire.vercel.app).

Displays custom order examples and lifestyle product photography with filter tabs.

## To add images

Replace each placeholder card's `<div class="card-img">` contents with an `<img>` tag:

```html
<img src="your-image.jpg" alt="Rush Shirts" style="width:100%;height:100%;object-fit:cover;">
```

## Deployment

Auto-deploys to Vercel on push to `main`.
